# AKS DevOps Practice Platform Documentation - සිංහල

මෙය AKS DevOps Practice Platform එක සඳහා official Sinhala documentation එකයි.

මෙම project එකෙන් learners සහ professionalsලාට පහත දේවල් practice කරන්න පුළුවන්:

- Terraform භාවිතයෙන් AKS platform provisioning
- CI/CD workflows
- GitOps workflows
- Gateway API routing
- Azure Container Registry සහ external registries
- Key Vault සහ Workload Identity
- Prometheus, Grafana, OpenTelemetry monitoring
- dev / qa / prod multi-environment patterns
