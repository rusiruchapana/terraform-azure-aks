# Translation Contributions

Official documentation is maintained in:

- English: `docs/en`
- Sinhala: `docs/si`

Community translations are welcome.

## Available translation starter folders

- te reo Māori: `docs/mi`
- Hindi: `docs/hi`
- Spanish: `docs/es`
- Portuguese: `docs/pt`
- French: `docs/fr`
- Arabic: `docs/ar`
- Indonesian: `docs/id`
- Vietnamese: `docs/vi`
- Japanese: `docs/ja`
- Korean: `docs/ko`
- Simplified Chinese: `docs/zh-CN`

## Translation rules

Please do not translate:

- CLI commands
- File names
- Folder names
- Terraform variable names
- Kubernetes YAML keys
- Resource names
- Code blocks

Please translate:

- Explanations
- Concepts
- Troubleshooting descriptions
- Beginner-friendly notes

## Example

Do not translate this code:

```yaml
nodeSelector:
  workload: user
```

You may translate the explanation around it.

## Review

For te reo Māori content, review from proficient te reo Māori speakers is especially welcome.

Thank you for helping make this project more accessible.
