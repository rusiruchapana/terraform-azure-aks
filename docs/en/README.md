# AKS DevOps Practice Platform Documentation

This is the official English documentation for the AKS DevOps Practice Platform.

This project helps learners and professionals practice:

- Terraform-based AKS platform provisioning
- CI/CD workflows
- GitOps workflows
- Gateway API routing
- Azure Container Registry and external registries
- Key Vault and Workload Identity
- Monitoring with Prometheus, Grafana, and OpenTelemetry
- Multi-environment dev / qa / prod patterns


## Hands-on labs
