# Examples

This folder contains example applications, CI/CD pipelines, GitOps patterns, secrets integrations, and observability configurations.

These examples are not deployed by the Terraform platform automatically. They are provided for learning, testing, and lab exercises.

## Categories

- `apps/` — sample applications
- `cicd/` — GitHub Actions, GitLab CI/CD, Azure DevOps, and Jenkins examples
- `gitops/` — Argo CD and Flux examples
- `secrets/` — Key Vault, Workload Identity, External Secrets, and CSI examples
- `observability/` — Prometheus, Grafana, and OpenTelemetry examples
