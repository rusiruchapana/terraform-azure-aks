# Observability Examples

This folder contains examples for monitoring and observing applications on AKS.

## Planned examples

- Prometheus and Grafana
- OpenTelemetry
- Application metrics
- Application traces
- Dashboards
- Alerts

## Learning purpose

Use these examples to practice:

- Reading cluster metrics
- Accessing Grafana safely
- Querying Prometheus
- Sending telemetry to OpenTelemetry Collector
- Building basic dashboards
- Creating basic alerts

## Important note

Monitoring services are kept internal by default.

Use port-forward for learning access unless a secure exposure pattern is configured.
