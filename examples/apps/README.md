# Application Examples

This folder contains sample application examples for AKS labs.

These examples are intentionally simple.

They are designed to help users understand how applications are packaged, deployed, exposed, and tested on AKS.

## Planned examples

- nginx
- dotnet-api
- nodejs-api

## Learning purpose

Use these examples to practice:

- Kubernetes Deployments
- Kubernetes Services
- Gateway API HTTPRoutes
- Container image references
- ACR and external registry image pulls
- Basic rollout verification
- Basic troubleshooting

## Important note

You are not limited to these sample applications.

After completing the examples, replace the sample app with your own application, Dockerfile, image, manifests, and deployment strategy.

This platform is app-agnostic.
