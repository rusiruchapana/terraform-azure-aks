# Professional Labs

Professional labs are for users who want to practice production-style platform engineering patterns on AKS.

These labs focus on GitOps, promotion, release strategies, incident response, and security hardening.

For the current full lab order, see:

    ../README.md

For the planned professional continuation, stack-specific labs, final projects, and AI-assisted learning track, see:

    ../../ROADMAP.md

## Track goal

Practice advanced AKS platform engineering workflows and production-style patterns.

## Current professional flow

1. Argo CD GitOps
2. Flux GitOps
3. dev to qa to prod promotion
4. Blue/green deployment
5. Canary deployment
6. Incident troubleshooting
7. Security hardening

## Planned professional continuation

The current professional flow should be completed first.

After that, the planned professional continuation includes:

- Node.js production deployment
- Python FastAPI production deployment
- .NET production deployment
- SonarQube or SonarCloud quality gates
- Dependency scanning
- Secrets scanning
- Infrastructure-as-Code security scanning
- Kubernetes policy validation
- SBOM generation
- Image signing
- Full quality gate pipeline

These topics are tracked in the project roadmap and should be added after the current professional flow is complete.

## Important note

These labs are advanced practice examples.

They are not complete production blueprints.

Use them to understand patterns, then adapt them to your own organization, risk model, and operational requirements.
